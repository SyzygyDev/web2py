Clarifications

Coed
There is no courtesy rule. Any player can serve to any player on the opposing team, including on game point.

Reverse Coed
It is a fault if a male attacks the ball in front of the attack line, completely above the height of the net, and the attack does not have an upward trajectory. Note that this is the only restriction. There is no restriction on serving or blocking.

Non-Hard Driven Ball Defense
At the first hit of a team, the ball can be doubled as long as it is one motion, unless played overhand with the fingers. If played overhand with the fingers, it cannot be doubled or caught.

Hard Driven Ball Defense
At the first hit of the team, the ball can be doubled as long as it is one motion. If played overhand with the fingers, it can be doubled as well as momentarily held or caught. Note that if the ball is played underhand with the fingers, it can be doubled but cannot be momentarily held or caught.

Serve Receive
The serve is never considered hard driven. If it is received overhand with the fingers, it must be clean and cannot be caught. Like any other first hit that is not from a hard driven ball, double contact is allowed as long as it is one motion (hands do not need to be together to receive serve).

Overhand Setting With Fingers
The serve is never considered hard driven. If it is received overhand with the fingers, it must be clean and cannot be caught. Like any other first hit that is not from a hard driven ball, double contact is allowed as long as it is one motion (hands do not need to be together to receive serve).
